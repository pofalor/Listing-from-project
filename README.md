# listing-from-project

Записывает содержимое всех файлов в заданной директории в Word-файл.

## Установка

1. Установите Python 3.
2. Создайте виртуальное окружение:

```powershell
py -m venv .venv
```

3. Активируйте виртуальное окружение:

```powershell
.\.venv\Scripts\activate
```

4. Установите зависимости:

```powershell
pip install -r requirements.txt
```

## Запуск

```powershell
python main.py
```

После запуска программа попросит:

- путь к директории, из которой нужно собрать листинг;
- имя выходного файла Word `.docx`;
- использовать ли двухколоночный режим.

Пример имени выходного файла:

```text
out.docx
```

## Игнорирование файлов

Файлы и папки, которые не нужно добавлять в документ, настраиваются в `.docignore`.

## Настройки оформления

Оформление документа настраивается в `listing_settings.json`.

- `document` - заголовок документа, шрифты, размеры и цвет заголовка/информационного блока.
- `layout.two_columns_spacing_inches` - расстояние между колонками в двухколоночном режиме.
- `files.include_appendix_word` - добавлять ли слово `ПРИЛОЖЕНИЕ` перед названием файла. По умолчанию `false`.
- `files.add_blank_line_after_file` - добавлять ли пустую строку после очередного файла. По умолчанию `false`.
- `one_column` - шрифты, размеры, цвета и интервалы для обычного режима.
- `two_columns` - шрифты, размеры, цвета и интервалы для двухколоночного режима.

Цвета задаются в формате `#RRGGBB`. По умолчанию названия файлов и основной текст черные.
